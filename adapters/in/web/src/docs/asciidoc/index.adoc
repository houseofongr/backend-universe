[[universe-api]]
== 유니버스(Universe) API

유니버스는 하오옹의 소리를 저장하는 공간으로, 내부에 여러 스페이스와 피스들을 담고있는 개념입니다.

[[space-api]]
== 스페이스(Space) API

스페이스는 유니버스 내부의 공간으로, 스페이스 내부에는 다른 스페이스 또는 피스가 존재할 수 있습니다.

operation::update-space-metadata[snippets='path-parameters,http-request,request-fields,http-response,response-fields']

[[overwrite-space-background-api]]
=== 스페이스 배경 덮어쓰기 API

스페이스의 배경 파일을 덮어쓰기합니다.

> 배경 파일 용량은 2MB를 초과할 수 없으며, 기존 파일은 삭제됩니다.

operation::overwrite-space-file[snippets='path-parameters,http-request,request-parts,http-response,response-fields']

[[move-space-api]]
=== 스페이스 이동 API

스페이스의 좌표를 다른 위치로 이동합니다.

> 두 점(시작, 종료좌표)을 이용해 사각형을 만들며, 내부의 다른 요소와 겹칠 수 없습니다.

operation::move-space[snippets='path-parameters,http-request,request-fields,http-response,response-fields']

[[delete-space-api]]
=== 스페이스 삭제 API

특정 스페이스를 삭제합니다.

> 스페이스 내부의 모든 연관된 정보(하위 스페이스, 피스, 사운드, 파일)를 함께 삭제합니다.

operation::delete-space[snippets='path-parameters,http-request,http-response,response-fields']
