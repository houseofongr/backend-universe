유니버스(Universe) API

유니버스는 하오옹의 소리를 저장하는 공간으로, 내부에 여러 스페이스와 피스들을 담고있는 개념입니다.

유니버스 트리

  • 유니버스는 다음과 같이 트리 형태로 구성됩니다.

  • 위 트리구조를 집합으로 표현하면 다음과 같습니다.

유니버스 생성 API

유니버스를 생성합니다.

HTTP request

POST /universes HTTP/1.1
Content-Type: multipart/form-data;charset=UTF-8; boundary=6o2knFse3p53ty9dmcQvWAIx1zInP11uCfbm
Host: api.houseofongr.com

--6o2knFse3p53ty9dmcQvWAIx1zInP11uCfbm
Content-Disposition: form-data; name=metadata
Content-Type: application/json

{
  "title": "우주",
  "description": "유니버스는 우주입니다.",
  "ownerID": "019812d4-73e3-7d80-9e8e-aa62af94c0b3",
  "categoryID": "019812d4-73e3-7b78-a49e-b8a68b70cfb6",
  "accessLevel": "PUBLIC",
  "hashtags": [
    "우주", "행성", "지구", "별"
  ]
}

Content-Disposition: form-data; name=thumbnail; filename=universe_thumb.png
Content-Type: image/png

유니버스 검색 API

유니버스를 검색합니다.

Path parameters

Table 1. /universes
Parameter Optional Description

page

true

보여줄 페이지 번호입니다.
* 기본값 : 1

size

true

한 페이지에 보여줄 데이터 개수입니다.
* 기본값 : 10

searchType

true

검색 방법입니다.
[CONTENT : 제목 + 내용 검색]
[AUTHOR : 작성자 검색]
[ALL : 작성자 + 제목 + 내용 전체 검색]

keyword

true

검색 키워드입니다.

sortType

true

정렬 방법입니다.
[TITLE : 제목 순]
[REGISTERED_DATE : 생성시간 순]
[VIEWS : 조회수 순] * 기본 정렬 : 생성시간 내림차순

isAsc

true

정렬 시 오름차순인지 여부입니다.

category

true

필터링할 카테고리 ID입니다.
